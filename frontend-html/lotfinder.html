Get your class schedule handy!

1. From the dropdown menu, choose the building your class is located in.-- (optional) you can give the class a name if you'd like (i.e., ENGL 161).

2. Click the green plus icon to append to your list of classes, and it'll show up in the box!

3. Repeat this for each of your classes (incl. lecture, discussion, lab, etc.).

4. Exclude any unwanted lots in the box to the right.

5. Upon completion, press the calculate button and scroll down to view a list of recommended parking lots and an interactive map with markers for your classes and recommended lots.

1. From the dropdown menu, choose the building your class is located in.-- (optional) you can give the class a name if you'd like (i.e., ENGL 161).

2. Click the green plus icon to append to your list of classes, and it'll show up in the box!

3. Repeat this for each of your classes (incl. lecture, discussion, lab, etc.).

4. Using the slider next to each class, rank each one (1 being the lowest priority) according to start time sensitivity--multiple classes can have the same rankings.

5. Exclude any unwanted lots in the box to the right.

6. Upon completion, press the calculate button and scroll down to view a list of recommended parking lots and an interactive map with markers for your classes and recommended lots.

Select lots you don't want to see in your calculation (e.g., they're currently full, no longer available to purchase, or just a lot you don't see yourself parking in).

Make sure to press confirm before calculating, otherwise all lots will be considered!
